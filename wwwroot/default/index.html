Welcome to SpacePack

Congratulations, when you see this page, you have successfully built a SpacePack on Docker!

SpacePack is in order to quickly deploy PHP environment, it contains the OpenResty, PHP, MariaDB, Memcached, Redis and phpMyAdmin.

You can check the environment: Proberv, phpinfo, Opcache and browse the phpMyAdmin page(The default port is 8080).

If have some problems, you can submit Issues or discuss with us on Telegram Group.

Good luck and have a good time with SpacePack <3
